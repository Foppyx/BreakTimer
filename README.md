This app will allow you to set up a simple timer to notify when you need to take a break from PC.

This is my first python app that I created because of back pain from sitting at the computer for a long time. Hope it will help you too!

 # Features
 
 - You can set a timer that will emit a sound when it finishes.
 - You can set the timer to restart automatically when the time runs out.
 - You can set it up so that a full-screen image pops up when the timer finishes.
 - You can import custom images and sounds.
 - The script will run in the background if closed, minimizing to the system tray.
